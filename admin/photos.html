3 Publish to the site

Two things go to the site: the optimized image files, and the gallery code that references them.

  1. A
    Click Download optimized images and move every downloaded .jpg and .webp into the site's assets/photos/ folder.
  2. B
    Open index.html, find the block between <!-- PHOTOS:START --> and <!-- PHOTOS:END -->, and replace everything between those two markers with the copied gallery HTML.
  3. C
    Commit / upload the changed index.html and the new files in assets/photos/. Done — the photos are live.

This tool mirrors optimize-photos.py (longest edge 1600px, EXIF rotation fixed, location data stripped, JPEG + WEBP output) so results match the command-line workflow. Everything runs locally in your browser — nothing is uploaded anywhere.
