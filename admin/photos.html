How do I get a token? (one time, ~2 minutes)
  1. Go to github.com/settings/personal-access-tokens/new (Fine-grained tokens).
  2. Token name: anything, e.g. "Photo Manager". Expiration: your choice.
  3. Resource owner: oaklandflyer.
  4. Repository access: choose Only select repositories → pick pittsburgh-shapers-website.
  5. Permissions → Repository permissions → Contents → set to Read and write.
  6. Click Generate token, copy it (starts with github_pat_), and paste it above. It's saved only in this browser.
