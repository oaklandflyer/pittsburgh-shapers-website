How do I get a token? (one time, ~2 minutes)
  1. Go to github.com/settings/personal-access-tokens/new (Fine-grained tokens).
  2. Resource owner: oaklandflyer. Repository access: Only select repositories → pittsburgh-shapers-website.
  3. Permissions → Repository permissions → Contents → Read and write.
  4. Generate the token (starts with github_pat_) and paste it above. It's saved only in this browser.
